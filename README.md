## Requirements
- Ruby (RubyGems)
- Node (npm)

## Install
1. `$ gem install bundler`
2. `$ bundle install`
3. `$ npm install`

## Build & Preview
- `$ gulp`

## Notice
- Filename: `_posts/:year/:month/YYYY-MM-DD-TITLE.md`
- Directory structure:

  ```
  _posts
  `-- 2016
      |-- 08
      `-- 09
          |-- 2016-09-11-first-post.md
          `-- 2016-09-12-second-post.md
  ```

- Permalink: `http://tech.ryukyu-i.co.jp/:year/:month/:day/:title/`
